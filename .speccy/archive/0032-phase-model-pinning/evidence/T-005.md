Probe loop iterates the six personas and, for each one, asserts both the rendered
`.codex/agents/reviewer-<persona>.toml` file and the
`resources/agents/.codex/agents/reviewer-<persona>.toml.tmpl` template carry the
expected `model = "gpt-5.5"` line and the expected
`model_reasoning_effort = "<tier>"` line. Twelve files × two assertions each =
24 distinct checks. The script emits one `FAIL:` line per missed assertion and
returns exit 1 if any failed; emits a single `PASS:` summary and returns exit 0
when all pass.

Probe command:

```bash
declare -A tier=(
  [business]=high
  [tests]=high
  [architecture]=high
  [security]=medium
  [style]=low
  [docs]=low
)
for persona in business tests architecture security style docs; do
  expected="${tier[$persona]}"
  for path in \
    ".codex/agents/reviewer-${persona}.toml" \
    "resources/agents/.codex/agents/reviewer-${persona}.toml.tmpl"; do
    grep -q '^model = "gpt-5.5"$' "$path" || echo "FAIL: ${path} missing model"
    grep -q "^model_reasoning_effort = \"${expected}\"\$" "$path" \
      || echo "FAIL: ${path} missing effort"
  done
done
```
