```
just reeject && just reeject && git diff --exit-code   # → IDEMPOTENT_CLEAN
git status --porcelain                                  # → (empty)
```

proving each new persona's source and ejected wrappers are consistent
and no ejected file was hand-edited. This is the
`speccy-work`/CI hygiene gate proof; the CHK is demonstrated by the
actual reeject-twice-then-diff command rather than by a unit test
(a Rust test cannot reach the `just`/MiniJinja eject pipeline in
isolation), so it is recorded here as the executed paper trail.
