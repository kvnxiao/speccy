# Evidence for SPEC-0060 T-002

Red-then-green paper trail for the `journal.prior_rounds`
attributes-only index (REQ-002). The change is additive and
pure-Rust: a new `JsonJournalBlockAttrs` struct plus
`to_json_journal_block_attrs` projection in `journal_show_output.rs`,
a `prior_rounds` field on `BundleJournal`, a partition in

new CLI flag and no behaviour observable only at the process
boundary, so the proof lives in three new integration tests under
`speccy-cli/tests/context.rs`. Red baseline is captured by
reverting the three `speccy-cli/src` files (keeping the new tests)
and observing the tests fail; green is the same tests against the
T-002 diff.
