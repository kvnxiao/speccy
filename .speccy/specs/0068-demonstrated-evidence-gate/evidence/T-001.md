# Evidence for SPEC-0068 T-001

Red-then-green paper trail for the demonstrated-evidence append-time gate.
Backs the `demonstrated` CHKs in T-001's roll call (CHK-001, CHK-002, CHK-003
under REQ-001; CHK-004, CHK-005, CHK-006 under REQ-002).

## Session 2026-06-27-attempt-1

The detection module (`demonstrated_chk_ids` / `scenario_heading_count`) and the
append-time gate in `append_under_lock` are new code; the red state is the
absence of any gate (an unbacked `demonstrated` claim was silently accepted).
The green state is the new module's unit tests plus the integration tests
driving the built `speccy` binary against scratch workspaces.

### Scenario 1 — Bullet-form unbacked claim is refused (CHK-001)

<red>
Before this task there was no gate: `speccy journal append --block implementer`
accepted any body, so a roll call bullet `- CHK-001 (...): demonstrated` with no
`evidence/T-NNN.md` was written unchanged. Nothing named the missing CHK or path.
</red>

<green>
`cargo test -p speccy-cli --test journal_append bullet_demonstrated_with_no_evidence_is_refused`
drives the binary with stdin `- CHK-001 (range parser): demonstrated -> evidence`
and no evidence file. The command exits non-zero, stderr contains both `CHK-001`
and `T-001.md`, and the journal file is never created:

```
running 1 test
test bullet_demonstrated_with_no_evidence_is_refused ... ok
```
</green>

### Scenario 2 — Prose-form unbacked claim is refused identically (CHK-002)

<red>
With no gate, the prose roll-call form `CHK-002 demonstrated by some_passing_test`
was equally accepted; the line-scoped heuristic that recognizes it did not exist.
</red>

<green>
`prose_demonstrated_with_no_evidence_is_refused` writes that prose body with no
evidence file. The command is refused identically — stderr names `CHK-002` and
`T-001.md`, no journal is created. The core unit test
`prose_form_roll_call_is_detected` confirms `demonstrated_chk_ids` parses the
prose form to `["CHK-002"]`:

```
test prose_demonstrated_with_no_evidence_is_refused ... ok
test parse::journal_xml::evidence::tests::prose_form_roll_call_is_detected ... ok
```
</green>

### Scenario 3 — Present-but-no-scenario evidence is refused distinctly (CHK-003)

</green>
